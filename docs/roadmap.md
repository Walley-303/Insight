# Roadmap

## Current State

All demos are browser-based HTML prototypes simulating the InSight dual-lens HUD experience. No hardware dependency required to present or iterate on the UX.

### Live Demos

| Demo | Path | Description |
|------|------|-------------|
| Scholar + Field | `scholar-field-demo/` | Combined dual-mode demo with all 8 scenarios |
| HUD | `hud-demo/` | Core HUD interface prototype |
| Mobile | `mobile-demo/` | Mobile form factor exploration |
| Landing Page | `landing-page/` | Public-facing product page |
| Unified Demo | `unified-demo/` | Consolidated demo experience |
| Presentation | `presentation/` | Pitch/demo deck format |
