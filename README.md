# InSight Technologies

> **See More. Know More. Say More.**

InSight is an AI-powered smart glasses platform that gives the wearer an invisible co-pilot — combining a voice-matched teleprompter, real-time AI response assistance, gaze-driven selection, scholar research tools, and field intelligence — all without anyone else knowing.

## Core Modes

### 🎤 Speaker & Debate Mode
- Voice-matched teleprompter scrolls word-by-word as you speak
- Off-script detection — last matched word highlighted in amber
- Live AI response suggestions with real-time web data
- Dual-eye gaze selection — right lens shows response cards, left lens activates selected response
- Multi-point debate queue with ambient dot tracking
- Wireless remote with scroll wheel, select, and record

### 📚 Scholar Mode
- Reads alongside you, connecting ideas across source material
- Smart quote clipping with auto-formatted citations (MLA, APA, Chicago, Bluebook)
- Write first, source later — AI agent scans academic databases for supporting material
- Works with PDFs, archival documents, and primary sources
- Integrates with JSTOR, arXiv, and PubMed

### 🔧 Field Mode
- Vehicle and unit analysis on sight — specs, market value, trade-in range, recall history
- Damage and defect flagging for insurance adjusters and mechanics
- Retail style assistant — reads customer aesthetic, suggests looks, checks live inventory
- POS and inventory system integration for enterprise users

### 📖 Reading Intelligence
- Eye tracking detects what you're reading
- Ask "what do they mean by that?" for instant context, definitions, and fact-checks
- Works on any physical or digital text

## Tech Stack

- **AI:** Claude (Anthropic) via API with real-time web search
- **Frontend:** HTML / CSS / JavaScript (vanilla), React components
- **Hardware target:** Waveguide AR lenses, IR eye tracking, 4-mic beamforming array, bone conduction audio
- **Connectivity:** Wi-Fi 6, Bluetooth 5.3
- **Companion app:** iOS & Android (planned)
